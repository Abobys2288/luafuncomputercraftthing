Window API

API.window(title, x, y, w, h)Create a window → returns win table
API.close(win)Destroy window
API.fitWindow(w, h)Get centered x, y, w, h for given size
API.redrawContent(win)Mark window content dirty (partial redraw)
API.getRenderer()Returns R (render module)
API.getScreenSize()Returns sw, sh

Drawing

API.drawText(x, y, text, fg, bg, maxW)Draw clipped text (maxW optional)
API.drawButton(x, y, w, h, text, pressed)W95-style button
API.clipText(text, maxW)Truncate text with ".."
R.fillRect(x, y, w, h, color)
R.drawLine(x1, y1, x2, y2, color)
R.drawW95Raised(x, y, w, h) / R.drawW95Sunken(x, y, w, h)
R.PALColor table: BLACK=0, WHITE=1, GRAY=2, ...

File System

API.readFile(path) / API.writeFile(path, content)
API.listFiles(path) / API.isDir(path)
API.joinPath(base, name) / API.getDir(path) / API.getFileName(path)
API.chooseFile(options, callback)File picker dialog
API.ensureDir(path)

Images (via shared library)

API.loadImage(path)→ pixels, w, h
API.loadAnimation(path)→ frames, w, h, delay, loop
API.detectImage(path)→ header table (no decode)

Network (async — never blocks UI)

net.init()Open modem → true/false
net.lookupAsync(name, timeout, callback)Find server without blocking
net.lookupSiteServerAsync(timeout, callback)
net.send(targetId, msg)Non-blocking send
net.siteListAsync(serverId, callback)
net.siteResolveAsync(serverId, name, callback)
net.siteGetAsync(hostId, name, path, callback)
net.discoverAsync(timeout, callback)
net.request(targetId, msg, expectedType, timeout, callback)Generic async request

Window Event Handlers

win.onDraw = function(win, cx, cy, cw, ch)Content area (cx,cy = top-left, cw,ch = size)
win.onKey = function(win, key, char)key = keys.xxx, char = typed char or nil
win.onClick = function(win, mx, my)Relative to content area (0,0 = top-left)
win.onScroll = function(win, dir, mx, my)dir = -1 up, +1 down
win.onDrag = function(win, dx, dy)Pan/drag delta
win.onDoubleClick = function(win, mx, my)
win.onClose = function(win)

Kernel / Crash Protection

API.try(label, fn, ...)pcall wrapper → routes to crash supervisor

API.rescue(reason)Close all windows, reload programs
